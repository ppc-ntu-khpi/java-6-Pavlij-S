## 📝 Завдання №12

Знайти рядок матриці розміром N x M з максимальною сумою елементів.

---

## 🛠️ Структура проєкту
Проєкт реалізовано за принципом розділення логіки (моделі) та демонстраційного інтерфейсу:
* `src/Main/java/domain/MatrixProcessor.java` — утилітарний клас, що містить чисту обчислювальну логіку (пошук індексу рядка з максимальною сумою).
* `src/Main/java/domain/Main.java` — тестовий клас, який містить статичну матрицю для демонстрації працездатності алгоритму та виведення результатів у консоль.

---

## 🚀 Тестові дані та демонстрація роботи

Для стабільності та повторюваності тестів у програмі використовується фіксована матриця $3 \times 3$:

| Рядок (Індекс) | Елементи | Сума рядка | Примітка |
| :---: | :---: | :---: | :---: |
| **Рядок 0** | `[3, 5, 2]` | 10 | |
| **Рядок 1** | `[12, 4, 9]` | **25** | ✨ **Максимальна сума** |
| **Рядок 2** | `[1, 0, 7]` | 8 | |

### Очікуваний результат:
Програма повинна визначити, що максимальну суму має рядок з індексом **1**, та вивести його вміст: `[12, 4, 9]`.
